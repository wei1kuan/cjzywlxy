# 文旅创意学院主页产品需求文档

## 1. 产品概述
文旅创意学院（荆楚非遗传承与创新实践中心）官方网站，展示学院特色、非遗文化传承与创新实践成果，面向学生、家长、企业和社会公众提供学院信息展示服务。
- 体现荆楚非遗文化与现代设计的融合，打造专业、现代、富有文化特色的学院形象
- 为招生宣传、专业展示、校企合作提供数字化平台

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 访客用户 | 无需注册 | 浏览学院信息、查看专业介绍、下载资料 |

### 2.2 功能模块
1. **主页**: 导航栏、轮播横幅、学院简介、新闻动态、特色项目、底部信息
2. **学院概况页**: 学院简介、领导团队、组织架构、学院荣誉
3. **党群工作页**: 党建工作、团学活动、工会活动
4. **专业设置页**: 7个专业介绍卡片，重点展示数字非遗设计与制作专业
5. **数字非遗专业详情页**: 专业介绍、特色课程展示（叶画、缠花、西兰卡普、雕花剪纸、汉绣、书法、漆扇、茶艺）
6. **教学科研页**: 教学成果、科研项目、学术交流
7. **学生天地页**: 学生活动、社团风采、获奖荣誉
8. **产教融合页**: 校企合作、实习基地、创新创业
9. **下载中心页**: 文件分类、资料下载

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**: 赭红色（荆楚文化特色）、深蓝色（稳重专业）、金色点缀（非遗文化）
- **辅助色**: 米白色、浅灰色、墨绿色
- **按钮样式**: 圆角按钮，带有文化纹样装饰，悬停时颜色加深
- **字体**: 标题使用方正清刻本悦宋（文化特色），正文使用思源黑体（现代清晰）
- **布局风格**: 卡片式布局，顶部导航，响应式网格系统
- **图标风格**: 结合非遗元素的扁平化图标

### 4.2 页面设计概述
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 主页 | 导航栏 | 固定顶部，赭红色背景，金色Logo，白色文字，移动端汉堡菜单图标 |
| 主页 | 轮播横幅 | 全宽轮播，深色遮罩，金色标题文字，自动播放3秒切换，左右箭头控制 |
| 主页 | 学院简介 | 左右布局，左侧配图（非遗元素），右侧文字介绍，赭红色装饰边框 |
| 主页 | 新闻动态 | 三列卡片网格，白色卡片背景，悬停阴影效果，赭红色标题 |
| 主页 | 特色项目 | 四列卡片网格，项目图片上方，名称和简介下方，悬停放大效果 |
| 主页 | 底部信息 | 深蓝色背景，白色文字，金色分隔线，联系方式居左，版权居右 |
| 专业设置页 | 专业列表 | 网格布局，数字非遗专业卡片更大更突出，金色边框装饰 |
| 数字非遗专业详情页 | 特色课程 | 八个课程卡片，每个课程配图和介绍，网格布局，悬停显示详情 |

### 4.3 响应式设计
- 采用桌面优先设计，移动端自适应
- 导航栏在移动端转为汉堡菜单
- 卡片网格在不同屏幕尺寸调整列数（桌面4列，平板2列，手机1列）
- 图片和文字比例自适应调整
- 触摸优化：按钮和链接尺寸适合触摸操作

### 4.4 动效设计
- 页面加载：模块依次淡入显示，营造层次感
- 轮播切换：平滑滑动过渡
- 卡片悬停：轻微上浮和阴影加深
- 导航切换：平滑滚动到对应区域
- 按钮点击：颜色过渡和轻微缩放
